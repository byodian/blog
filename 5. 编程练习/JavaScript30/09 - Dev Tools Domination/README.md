## 知识点 - console 方法

### console.log()

- %s 变量

```js
console.log('I am a laugher %s', '😊'); // I am a laugher 😊
```

- %c CSS 样式

```js
console.log('%c I am some great text', 'color: red; font-size: 30px;')
```
