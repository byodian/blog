### splice()

语法：arr.splice(index, [deletCount, elems, ..., elemN])

从 index 开始：删除 deleteCount 元素并在当前位置插入 elem1, ..., elemN。最后返回已删除元素的数组。

### slice()

语法：arr.slice(start, end)

从元素的开始索引 "start" 复制到 "end" (不包括 "end") 返回一个新的数组。
